🎯 Joba haqqında
Qaraqalpaq urıwları — bul qaraqalpaq urıwlariń tolıq shejiresin cifrlıq formatta saqlawǵa arnalǵan veb-joba. Joba 50+ urıw, 300+ tiyre/kóshe haqqında maǵlıwmatlardı qamaydı.
Tiykarǵı maqsatlar:
📚 Qaraqalpaq urıwlıq qurılımın saqlaw
🔍 Uruwlıq tegdi tez anıqlaw
⚖️ Ekzogamiya erejelerin tekseriw
🎓 Gılımı-izertlew jumıslarına qollap-quwatlaw

✨ Imkániyatlar

| Imkániyat                  | Sıpatlaması                           |
| -------------------------- | ------------------------------------- |
| 🌳 **Interaktiv sxema**    | D3.js tiykarındaǵı urıwlıq teregi     |
| 📋 **Tizimlik kórinis**    | Ierarxiyalıq tizim izlew júyesi menen |
| 💑 **Ekzogamiya tekseriw** | Neke ruqsatlılıǵın avtomat tekseriw   |
| 🎨 **Adaptiv dizayn**      | Barlıq qurılmalar ushın               |
| 🌙 **Qarańǵı tema**        | Kózge ǵaylı interfeys                 |
| 🔄 **Ush til**             | Qaraqalpaqsha + Qazaqsha + Orıssha    |


🛠️ Texnologiyalar

Frontend:
├── HTML5 + CSS3
├── JavaScript (ES6+)
├── D3.js (vizualizatsiya)
└── SVG (grafika)

Dizayn:
├── CSS Variables (temizatsiya)
├── Flexbox + Grid
└── Mobile-first approach

⚙️ Ornatıw

# Repozitoriydi klonlaw
git clone https://github.com/username/qaraqalpaq-uruwlari.git

# Joba buqasına ótiw
cd qaraqalpaq-uruwlari

# Lokal serverdi iske túsirw (mısalı, VS Code Live Server)
# Yamasa qarapayım HTTP server:
python -m http.server 8000

Talaplar:
Zamanawı brauzer (Chrome, Firefox, Safari, Edge)
JavaScript qosılǵan bolıwı kerek

🎯 О проекте
Qaraqalpaq urıwları — это веб-проект, предназначенный для сохранения полной родословной каракалпакских родов в цифровом формате. Проект включает данные о 50+ родах, 300+ тийре/көше.
Основные цели:
📚 Сохранение каракалпакской родовой структуры
🔍 Быстрое определение родового происхождения
⚖️ Проверка правил экзогамии
🎓 Поддержка научно-исследовательской работы

✨ Возможности

| Возможность                    | Описание                                 |
| ------------------------------ | ---------------------------------------- |
| 🌳 **Интерактивная схема**     | Родовое дерево на базе D3.js             |
| 📋 **Списковое представление** | Иерархический список с поиском           |
| 💑 **Проверка экзогамии**      | Автоматическая проверка разрешения брака |
| 🎨 **Адаптивный дизайн**       | Для всех устройств                       |
| 🌙 **Тёмная тема**             | Удобный для глаз интерфейс               |
| 🔄 **Три языка**               | Каракалпакский + Казахский + Русский     |


🛠️ Технологии

Frontend:
├── HTML5 + CSS3
├── JavaScript (ES6+)
├── D3.js (визуализация)
└── SVG (графика)

Дизайн:
├── CSS Variables (темизация)
├── Flexbox + Grid
└── Mobile-first approach
